# libVLC Android SDK

libVLC for Android (arm64-v8a, armeabi-v7a, x86, x86_64)

compiled with flags:  -l -r --no-ml

## Import

Add this to your project's `build.gradle`.
```groovy
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```


Add this to you app's `build.gradle`:

```groovy
dependencies {
    implementation 'com.github.borut123:libvlc_android_sdk:3.0.11'
}
```
